# Products App — Front-End Test

Mini aplicación SPA para comprar dispositivos móviles, desarrollada como prueba técnica de Front-End.

> Este README se irá completando de forma incremental a medida que el proyecto avance (arquitectura, scripts, decisiones técnicas, gestión de caché, etc.).

## Estado

🚧 En desarrollo — ver [CHECKLIST.md](./CHECKLIST.md) para el detalle de requisitos y su estado de implementación.

## Stack

- React + Vite (sin TypeScript)
- React Router
- Context API
- SCSS Modules
- ESLint (Flat Config)

## API

- `GET /api/product` — listado de productos
- `GET /api/product/:id` — detalle de producto
- `POST /api/cart` — añadir producto a la cesta

Dominio: `https://itx-frontend-test.onrender.com/`
